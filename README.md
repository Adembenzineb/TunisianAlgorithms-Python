# TunisianAlgorithms-Python 🇹🇳 🐍

This repository contains a collection of Python implementations for algorithmic problems studied in Tunisian schools and programming curricula. It provides solutions to common programming challenges with detailed explanations.

## 🚀 Overview

TunisianAlgorithms-Python is designed to help students and educators understand and implement fundamental algorithms taught in Tunisian educational institutions. Each algorithm is implemented with clear code and comprehensive explanations to facilitate learning.

## 📂 Repository Structure

The repository is organized as follows:

- **/algorithms** 📝: Contains the Tunisian algorithm pseudocode files (`.algo` or `.txt` format)
- **/implementations** 💻: Contains the Python implementations of each algorithm
  - Each implementation is paired with its corresponding pseudocode
- **/examples** 🧪: Sample usage and test cases for each algorithm
- **/docs** 📚: Additional documentation and explanatory materials

## 🗂️ Content Categories

- **Basic Algorithms** ⚙️: Implementation of fundamental algorithms taught in early programming courses
- **Data Structures** 🏗️: Common data structures and their operations
- **Sorting Algorithms** 📊: Various sorting techniques with complexity analysis
- **Search Algorithms** 🔍: Different search methods with examples
- **Graph Algorithms** 🕸️: Graph representation and traversal techniques
- **Dynamic Programming** 🧩: Solutions to optimization problems
- **Numerical Methods** 🔢: Mathematical algorithms commonly used in scientific computing

## 🛠️ Technical Requirements

- Python 3.8+ recommended 🐍
- No external dependencies required for basic implementations ✅
- Some advanced algorithms may require NumPy or other scientific libraries 📈

## 📁 File Structure Example

For each algorithm, you'll find:

```
/algorithms/recherche_dichotomique.algo  # Tunisian pseudocode 📝
/implementations/recherche_dichotomique.py  # Python implementation 💻
/examples/recherche_dichotomique_examples.py  # Usage examples 🧪
```

## 🔤 Tunisian Algorithm Pseudocode Commands

In Tunisian educational contexts, algorithm pseudocode often uses specific French/Arabic-influenced commands and syntax. Below are the comprehensive list of pseudocode commands used in Tunisian algorithm courses and their Python equivalents:

| Tunisian Pseudocode | Python Equivalent | Description |
|-------------------|-------------------|-------------|
| `Algorithme` ✨ | Function definition | Defines the algorithm name |
| `Début` 🏁 | Start of code block | Beginning of algorithm/function |
| `Fin` 🏆 | End of code block | End of algorithm/function |
| `Variables` 📦 | Variable declarations | Declares variables used |
| `Const` 🔒 | Constants | Defines constant values |
| `Si ... Alors ... Sinon ... FinSi` 🔀 | `if ... elif ... else` | Conditional statements |
| `Si ... Alors ... Sinon Si ... Sinon ... FinSi` 🔀 | `if ... elif ... else` | Extended conditionals |
| `Pour ... à ... Faire ... FinPour` 🔄 | `for i in range()` | For loop with range |
| `Pour ... à ... Pas ... Faire ... FinPour` 🔄 | `for i in range(start, end, step)` | For loop with step |
| `Pour Chaque ... Dans ... Faire ... FinPourChaque` 🔄 | `for item in collection` | Iterating through collections |
| `Tant Que ... Faire ... FinTantQue` 🔁 | `while` | While loop |
| `Répéter ... Jusqu'à` 🔁 | `while not` | Do-while equivalent |
| `Lire` 📥 | `input()` | Read input |
| `Écrire` 📤 | `print()` | Output to console |
| `Afficher` 📤 | `print()` | Alternative output command |
| `Retourner` 🔙 | `return` | Return a value |
| `Fonction` 🧮 | `def` | Define a function |
| `Procédure` 📋 | `def` (no return) | Define a procedure |
| `Tableau` 📊 | Lists/Arrays | Array data structure |
| `Matrice` 🧮 | 2D Lists/Arrays | 2D array structure |
| `Enregistrement` 📝 | `class` or dictionary | Record/struct equivalent |
| `Type` 🏷️ | Custom types/classes | Type definition |
| `Et` 🔗 | `and` | Logical AND |
| `Ou` 🔗 | `or` | Logical OR |
| `Non` ❌ | `not` | Logical NOT |
| `Mod` 🔢 | `%` | Modulo operator |
| `Div` 🔢 | `//` | Integer division |
| `Appel` 📞 | Function call | Call a procedure/function |
| `Selon ... Cas ... Autre ... FinSelon` 🔄 | `match/case` or `if/elif` chains | Switch-case equivalent |
| `Vrai` ✅ | `True` | Boolean true value |
| `Faux` ❌ | `False` | Boolean false value |
| `Null` 🚫 | `None` | Null value |
| `Arrêter` 🛑 | `break` | Exit a loop |
| `Continuer` ⏭️ | `continue` | Skip to next iteration |
| `En-tête` 📋 | Function signature | Defines function parameters |
| `Déclarer` 📝 | Variable declaration | Explicitly declare a variable |
| `Entier` 🔢 | `int` | Integer type |
| `Réel` 🔢 | `float` | Floating-point type |
| `Caractère` 🔤 | `str` (single char) | Character type |
| `Chaîne` 📝 | `str` | String type |
| `Booléen` ✅❌ | `bool` | Boolean type |
| `Taille` 📏 | `len()` | Get length/size |
| `Concat` 🔗 | `+` (for strings) | String concatenation |
| `SousAlgorithme` 🧩 | Helper function | Subroutine definition |
| `Chercher` 🔍 | Search functions | Generic search operation |
| `Trier` 📊 | Sorting functions | Generic sort operation |
| `Échanger` 🔄 | Swap operation | Exchange two values |
| `AllerÀ` ↪️ | Not recommended in Python | Goto equivalent (discouraged) |
